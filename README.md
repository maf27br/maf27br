- **[Mestrado — Censo Educacional](https://github.com/maf27br/Mestrado_Censo_Educacacional)** —
  análise de indicadores da pesquisa de mestrado, em **R**.

## 💼 Desenvolvimento de Software

Portfólio de produtos SaaS em desenvolvimento _(repositórios privados)_:

- **Hub de controle (control plane)** — cadastros, assinaturas, onboarding e mensageria;
  integração de pagamentos (Asaas), multi-tenant e conformidade LGPD. **Go + SQLite**.
- **CRM de carteira de clientes** — gestão ponta a ponta para corretores, do lead ao contrato.
  **Go + Vue 3**.
- **Gestão para pequenos negócios (PWA)** — catálogo, estoque e vendas mobile-first, com
  funcionamento offline. **Go + Vue 3 + PWA**.

> Infraestrutura em **Docker** sobre **Oracle Cloud**, com deploy e observabilidade próprios.
